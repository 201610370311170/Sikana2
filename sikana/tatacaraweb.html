Tata Cara Penggunaan Website :

1. Beranda yaitu halaman utama dari sebuah website kami yang dirancang sebagai pusatnya. Beranda kami memuat latar belakang website kami, Jadwal Kegiatan Program Kesehatan Ibu dan Anak, dan Alur Kesehatan untuk Ibu Hamil.

2. Periksa yaitu halaman yang berisikan rekapan data-data pasien untuk memudahkan petugas membuat sebuah riwayat pasien, dan dilengkapi rujukan atau cek kesehatan yang lainnya.

3. Tata cara website yaitu halaman untuk para pengguna supaya memahami alur atau fungsi website kami.

4. Masuk yaitu halaman untuk para pengguna dalam pendaftaran sebagai anggota. Dan login untuk admin menggunakan Username dan login untuk petugas menggunakan NIP petugas.
